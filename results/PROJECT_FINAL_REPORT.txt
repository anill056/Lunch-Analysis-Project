1. PROJECT SUMMARY
================================================================================
This project aimed to develop an AI-supported classification system using user 
scores and food images to analyze the quality of food served in the Mudanya University 
cafeteria and suggest improvements. 
By providing an analysis in terms of user satisfaction and visual presentation
features of the meals, data-driven recommendations were presented to increase the service
quality of the cafeteria.



================================================================================
2. METHODS USED
================================================================================

2.1. Data Collection
--------------------------------------------------------------------------------
• User Surveys: 187 survey responses collected
• Food Images: 64 food photos analyzed
• Number of Categories: 8 different food categories
• Total Score Count: 2945 user scores

2.2. Deep Learning Model
--------------------------------------------------------------------------------
• Model Type: Convolutional Neural Network (CNN)
• Transfer Learning: MobileNetV2 (ImageNet weights)
• Image Size: 224x224 pixels
• Training Epochs: 35 epochs
• Optimization: Adam optimizer
• Data Augmentation: Rotation, Flip, Zoom techniques applied

2.3. Evaluation Metrics
--------------------------------------------------------------------------------
• Accuracy: Overall success rate of the model
• Precision: Accuracy rate of positive predictions
• Recall: Rate of finding true positives
• F1-Score: Harmonic mean of Precision and Recall
• Confusion Matrix: Detailed performance analysis by category

================================================================================
5. RECOMMENDATIONS
================================================================================

5.1. Food Quality Improvements
--------------------------------------------------------------------------------
• Recipe and presentation improvements should be made in low-performing categories
• Spice balance should be optimized according to user feedback
• Visual presentation quality should be increased (plate layout, garnish use)

5.2. Data Collection and Analysis
--------------------------------------------------------------------------------
• Model performance can be increased by collecting more food images
• User satisfaction should be tracked with regular surveys
• Seasonal changes and special days should be analyzed

5.3. System Development
--------------------------------------------------------------------------------
• A real-time feedback system should be established with a web/mobile app
• A system for automatic visual quality control should be integrated
• A predictive menu planning system could be developed

================================================================================
6. CONCLUSION
================================================================================

Within the scope of this project, the quality of food served in the university
cafeteria was successfully analyzed using deep learning techniques. 
An accuracy rate of 82.00% was achieved using the MobileNetV2 model with a transfer learning approach.
The combination of user scores and visual analysis allowed the objective evaluation of food quality.  
The obtained results offer concrete recommendations to increase the service quality of the cafeteria. 
This study serves as an important example of how AI-supported systems can be used in the food service sector.
